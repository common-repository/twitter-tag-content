=== Twitter Tag Content ===
Contributors: hughwillfayle
Donate Link: http://hughwillfayle.de
Tags: content, post, page, twitter, hashtags
Requires at least: 2.0.2
Tested up to: 3.0.4
Stable tag: 0.3

This plugin replaces words with a leading "#" with a link to the twitter hashtag search, or to a user by giving a leading @

Currently, there is no support for special chars

== Installation ==

1. Upload `twitter-tag-content.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
